About the Flatsheet editor demo

Flatsheet is a tool for managing tabular data with a friendly editor.

The current goals are to improve the editor UI to create the simplest possible editing experience, to create integrations with other tools, like syncing with Google Spreadsheets and dat, and to allow group collaboration on a dataset through the UI.

This editor is a demo of the new version of Flatsheet that's being developed with the support of an OpenNews code sprint grant.

Get the code, submit issues, get involved

The code for this project lives on GitHub at github.com/flatsheet/editor. Please submit issues to our issues queue: github.com/flatsheet/editor/issues.
